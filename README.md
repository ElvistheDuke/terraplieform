# Terraplieform

A modern web application form for getting data for the Terraplie Demo Program.

## Features

- **Responsive Form**: Its a form focused on the UX to make filling it feel easy
- **Email Result**: The results of the form are sent to admins email
- **Admin Dashboard**: There's an Admin Dashboard for showing analytics of the users based on various criterias

## Technologies Used

- Nextjs
- TailwindCss
- Resend Api (for sending mails)
- ShadCN UI
- Framer Motion
- ReCharts

## Getting Started

### Prerequisites

- Node.js 16+
- npm or yarn

### Installation

```bash
git clone https://github.com/ElvistheDuke/terraplieform.git
cd terraplieform
npm install
```

### Usage

```bash
npm start

npm run dev
```

Visit `http://localhost:3000` in your browser.
